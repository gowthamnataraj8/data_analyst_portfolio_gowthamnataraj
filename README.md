# Sales Performance Analysis using SQL

## Project Objective
Analyse sales data to identify trends, top products, top customers and best performing countries.

## Business Questions
1. What is the total sales by year?
2. Which are the top 5 products by revenue?
3. Which country generated the highest sales?
4. What is the monthly sales trend?
5.Who are the top customers?

## Tools Used
- SQL (MySQL)
- CSV Dataset

## Dataset
Sample Sales Data from Kaggle

## Files in this Project
- sales_analysis.sql → SQL queries for analysis  
- sales_data_sample.csv → Dataset used for analysis
